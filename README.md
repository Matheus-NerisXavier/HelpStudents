# HelpStudents - Comunidade de Estudos Integrada

Bem-vindo ao **HelpStudents**, uma plataforma moderna e dinâmica feita para facilitar a jornada de alunos e universitários.

## 🚀 Funcionalidades Atuais
- **Página Inicial Premium:** Design youthful e moderno com foco em experiência do usuário.
- **Estrutura de Rotas:** Preparada com React Router para expansão rápida.
- **Sistema de Design:** Baseado em Glassmorphism com paleta de cores vibrantes.

## 📅 Próximas Fases
- [ ] **Autenticação com Supabase:** Login seguro e perfis de usuário.
- [ ] **Fóruns Colaborativos:** Espaço para tirar dúvidas e compartilhar conhecimento.
- [ ] **Mural de Fotos:** Compartilhamento de lousas e cadernos (Storage Supabase).
- [ ] **Módulo de Mobilidade:** Solicitação e oferta de caronas.
- [ ] **Módulo de Moradia:** Divisão de hospedagem e repúblicas.

## 🛠️ Tecnologias Utilizadas
- **React 18** (Frontend)
- **Vite** (Build tool)
- **Lucide React** (Ícones)
- **Framer Motion** (Animações)
- **Supabase** (Backend - Próximo passo)

## 📦 Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Rode o servidor de desenvolvimento: `npm run dev`
4. Acesse em: `http://localhost:5173`
